# teluq-sci1402

Projet : Analyse Stratégique du Bassin LinkedIn - Bionest
Ce projet utilise la science des données pour optimiser le recrutement d'un poste de Direction de l'Ingénierie.

Algorithme de Scoring : Pour évaluer le bassin de 184 candidats, un algorithme de scoring sur 100 points a été développé selon les poids suivants :

1. Membre OIQ (40 pts) : Conformité légale indispensable pour la signature de plans.
2. Expérience de Gestion (30 pts) : Capacité à diriger l'équipe actuelle.
3. Localisation (30 pts) : Proximité (< 30 min) des bureaux de Shawinigan.

Segmentation des Profils

Les candidats sont classés en 4 catégories pour faciliter la prise de décision d'affaires :
1. Profils Idéaux : Score de 100.
2. Experts Mobiles : Ingénieurs qualifiés mais hors-zone géographique.
3. Relève Technique : Ingénieurs locaux nécessitant un coaching en gestion.
4. Bassin Secondaire : Profils nécessitant des compromis majeurs.
